# 🎬 FukulisanE AI Studio

AI-powered business operating system for content creators.

## Quick Start

```bash
bun install && bun run dev
```

## Docker

```bash
docker compose up -d
```

## Features

- 🎥 Multi-camera studio with WebRTC
- 📡 Livestream with RTMP ingest
- 🎙️ Podcast management
- ✨ AI content generation
- 🧠 Business intelligence
- ⚡ Workflow automation

## Tech Stack

React, Vite, Tailwind, Hono, Prisma, SQLite, WebRTC, Docker
